| 属性 | Description |
| :--- | :--- |
| htmlBody   | 用于邮件正文的 HTML。 |
| subject    | 用于邮件的主题。 |
| importance | 用于邮件的重要性标志：`low`、`normal` 或 `high`。 |
| toRecipients | 添加到邮件“收件人”字段的用分号 (;) 分隔的电子邮件地址字符串。 |
| ccRecipients | 添加到邮件“Cc(抄送)”字段的用分号 (;) 分隔的电子邮件地址字符串。 |
| bccRecipients | 添加到邮件“Bcc(密件抄送)”字段的用分号 (;) 分隔的电子邮件地址字符串。 |
