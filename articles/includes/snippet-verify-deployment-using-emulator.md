使用 [Bot Framework Emulator](~/bot-service-debug-emulator.md) 验证机器人部署。 

将机器人的消息终结点输入到模拟器的地址栏。 如果使用 Bot Framework SDK 构建机器人，则终结点应当以 */api/messages* 结尾。

示例： `https://<appname>.azurewebsites.net/api/messages`
